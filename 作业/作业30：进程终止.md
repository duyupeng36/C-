
# 作业30：[[进程终止]]


如果子进程调用 `exit(-1)`，父进程将会看到何种退出状态（由 `WEXITSTATUS()`返回）？

```c

#include <unistd.h>
#include <sys/wait.h>

#include "base.h"

int main() {

    switch (fork()) {
        case -1:
            errExit("fork");
        case 0:
            exit(-1);
        default:
            sleep(1);
            int status;
            wait(&status);
            printf("status=%d\n", WEXITSTATUS(status)); // 255
    }
}
```

由于 `exit(status)` 的参数 `status` 只有低 $8$ 位被父进程使用。由于 `-1` 在计算机内存中采用补码编码，低 $8$ 位为 `11111111`，该值为 $255$
